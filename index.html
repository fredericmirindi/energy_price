API Documentation

RESTful API

Access our energy price forecasting models through our powerful REST API. Get real-time predictions, historical data, and model insights.

Base URL: https://api.energyforecast.ai/v1
GET /api/v1/price/current

Get current electricity price

{
  "price": 45.23,
  "currency": "USD/MWh",
  "timestamp": "2025-07-26T05:38:00Z",
  "region": "NYISO"
}
GET /api/v1/forecast/{hours}

Get price forecast for specified hours

{
  "forecast": [45.23, 46.12, 47.35],
  "hours": 24,
  "model": "transformer",
  "confidence": 0.942
}
POST /api/v1/predict

Make custom prediction with parameters

{
  "model": "transformer",
  "region": "NYISO",
  "hours": 24,
  "features": {...}
}
GET /api/v1/models

List available ML models

{
  "models": [
    {"name": "transformer", "accuracy": 0.942},
    {"name": "lstm", "accuracy": 0.925}
  ]
}

About the Project

Project Overview

This project leverages state-of-the-art machine learning algorithms to forecast electricity prices with high accuracy. By analyzing historical market data, weather patterns, and economic indicators, our models provide reliable predictions for energy traders, utilities, and consumers.
